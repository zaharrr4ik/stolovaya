Проект: Веб-приложение для управления школьной столовой
Описание проекта
Веб-приложение для автоматизации управления питанием в школьной столовой. Система обеспечивает учет посещаемости, управление меню, контроль баланса учащихся и автоматизацию заявок на закупку продуктов.
Функциональные возможности
Роли пользователей
1. Ученик
Просмотр меню на неделю
Оплата питания (завтраки/обеды)
Просмотр баланса
Оставление отзывов о блюдах
Редактирование профиля (аллергии, предпочтения)
2. Повар
Управление меню на неделю
Учет продуктов на складе
Подача заявок на закупку
Учет выданного питания
Управление готовыми блюдами
3. Администратор
Просмотр статистики по питанию
Управление заявками на закупку
Генерация отчетов
Просмотр отзывов учащихся
Финансовый контроль
Технологический стек
Backend

Jinja2 - шаблонизатор
База данных
SQLite3 - встраиваемая СУБД
SQLAlchemy (опционально) - ORM
Frontend
HTML5 - структура страниц
CSS3 - стилизация
JavaScript (минимальный) - интерактивность
Инструменты разработки
Git - контроль версий
GitHub - хостинг кода
Архитектура системы
Трехзвенная архитектура

Слой данных (SQLite база данных)
